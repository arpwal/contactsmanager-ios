# Contacts Manager for Businesses - iOS Package

[![Swift](https://img.shields.io/badge/Swift-5.5+-orange.svg)](https://swift.org)
[![iOS](https://img.shields.io/badge/iOS-17.0+-blue.svg)](https://developer.apple.com/ios/)
[![Version](https://img.shields.io/badge/version-1.38.0-green.svg)](https://github.com/arpwal/contactsmanager-ios/releases)

A powerful contacts management framework for iOS business applications.

## Installation

### Swift Package Manager

Add the following to your `Package.swift` dependencies:

```swift
dependencies: [
    .package(url: "https://github.com/arpwal/contactsmanager-ios.git", .exact("1.38.0"))
]
```

Or in Xcode:

1. File > Add Packages
2. Enter Package URL: `https://github.com/arpwal/contactsmanager-ios.git`
3. Select "Exact Version" and enter "1.38.0"

## Features

- Comprehensive contact management
- Business-focused contact organization
- iOS 17+ support
- Swift 5.5+ compatibility

## Requirements

- iOS 17.0+
- Swift 5.5+

## Usage

Import the framework in your Swift file:

```swift
import ContactsManager
```
